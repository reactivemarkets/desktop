# describe `[get]`

Prints the current details of an instance.

## Parameters

#### &ndash;&ndash;uid `[uid]`

The instance identifier.

#### &ndash;&ndash;output, -o `[choices: "yaml", "json"]` `[default: "yaml"]`

The output format.
