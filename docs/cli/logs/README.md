# logs

Fetch the logs of an instance.

## Parameters

#### &ndash;&ndash;follow, -f

Follow log output.

#### &ndash;&ndash;output, -o `[choices: "yaml", "json"]` `[default: "json"]`

The output format.

## Examples

<code-group>
<code-block title="Get logs">

```bash
desktop logs
```

</code-block>
<code-block title="Follow logs">

```bash
desktop logs -f
```

</code-block>
</code-group>
